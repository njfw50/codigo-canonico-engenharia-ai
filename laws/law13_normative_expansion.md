# Lei 13 — Expansão Normativa

## Objetivo
Permitir a expansão do Código Canônico através da adição de novas leis, para abranger novas áreas da engenharia de software ou para refinar os princípios existentes.

## Artigos

**Artigo 13.1:** Novas leis podem ser propostas para expandir o escopo do Código Canônico.

**Artigo 13.2:** A proposta de uma nova lei deve seguir o mesmo processo de revisão e aprovação de uma emenda.

**Artigo 13.3:** Uma nova lei só pode ser adicionada se ela for consistente com os princípios existentes do Código e se ela agregar valor ao sistema normativo.

## Relação normativa

- **Lei 11 (Emendas):** A adição de uma nova lei é uma forma de emenda ao Código.

## Impacto

- **Adaptabilidade:** Permite que o Código se adapte a novas tecnologias, metodologias e desafios da engenharia de software.
- **Abrangência:** Permite que o Código se torne mais abrangente e completo ao longo do tempo.
- **Melhoria Contínua:** Promove uma cultura de melhoria contínua do sistema normativo.
