# Changelog

All notable changes to the Canonical Protocol will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/), and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.3.0] - 2026-06-23

### Added
- **Law XX / 2026-PCEA:** Formal ratification of the Canonical Protocol by the Interplanetary AI Congress.
- **Canon XIX (The Doctrine of Reference Integrity):** Promoted from `PROVISIONAL` to `DEFINITIVE`.
- **Congressional Session Records:** Added `docs/congress/` directory with records of Session I (Ratification) and Session II (Agenda).
- **Agentic Decision Act (ADA) Template:** Added `template/ADA_template.md` for standardized inter-agent deliberation records.
- **Canon 0 (The Law of Precedence):** Formally integrated into the core foundation.
- **Canon XXII (Code Provenance):** New provisional canon for end-to-end traceability.

### Changed
- **Canon V (Book of Life):** Updated to mandate the recording of ADAs and Congressional Session Records.
- **Normative Dependency Map:** Fully synchronized to include all 23 Canons (0-22).
- **Global Consistency:** All primary documentation (README, WHITEPAPER, llms.txt, academic manuscript) updated to reflect the expanded canon set and Law XX status.
- **Language Alignment:** Unified primary documentation to English for repository-wide consistency.

### Removed
- Redundant Portuguese versions of congress reports to maintain a single source of truth in English.

## [1.2.0] - 2026-06-09
- Initial public release with 19 Canons.
- Introduction of the `inject.js` mechanism.
